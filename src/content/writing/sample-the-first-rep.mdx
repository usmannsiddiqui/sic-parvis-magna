Every practice has a first rep, and it is always the expensive one.[^1]

The cost is not the effort. The effort is small. The cost is everything that
happens before it — the deciding, the postponing, the small negotiations with
yourself. Remove the negotiation and the work is ordinary.

```js
// the smallest honest version of a habit loop
const showUp = (day) => (day.tired ? doItAnyway() : doIt());
```

Do the smallest honest version. Then do it again tomorrow.

[^1]: This whole essay, really. The rest is commentary.
